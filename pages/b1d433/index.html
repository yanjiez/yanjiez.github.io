微信小程序 启动参数(onLaunch options query)自动转换为小写

# 现象

分享链接参数使用驼峰，如：memberId

m2用户点击分享链接进入小程序，options.query.memberId 为 undefined

# 原因

onLaunch options query 是小程序初始化时传入的参数，由于小程序对于参数的处理是不区分大小写的，所以为了统一，都会自动转换成小写

# 解决

使用小写参数名，如member_id

# 补充

另外需要注意，与onLaunch options query不同的是，router options 是页面跳转时传入的参数，大小写必须与实际的参数名一致，才能正确获取到参数值。
