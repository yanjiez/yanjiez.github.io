消除ios点击按钮时出现的一个半透明的灰色背景

# 解决

body {
    -webkit-tap-highlight-color: rgba(0, 0, 0, 0)
}
1
2
3

注：webkit是苹果浏览器引擎，tap点击，highlight背景高亮，color颜色，颜色用数值调节。
